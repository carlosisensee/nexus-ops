    Acesse o Google Tag Manager

    Vá em tagmanager.google.com e selecione seu contêiner.

  2. 2

    Crie uma nova tag

    Clique em Tags → Nova. Escolha o tipo HTML Personalizado.

  3. 3

    Cole o código gerado

    Copie o código da aba "Código GTM" e cole no campo HTML. Marque "Suporte para document.write" como desmarcado.

  4. 4

    Configure o acionador

    Selecione All Pages ou crie um acionador customizado para páginas específicas.

  5. 5

    Publique e teste

    Clique em Enviar para publicar. Use o modo Preview do GTM para validar antes de ir ao ar.
